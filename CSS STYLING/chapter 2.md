# CSS Properties Reference & Explanation

This guide collects common CSS properties in one place with explanations and examples.

---

## 1Ô∏è‚É£ Text & Font Properties

| Property | Description | Example |
|----------|------------|---------|
| `color` | Sets the text color | `color: red;` |
| `font-size` | Sets the size of the text | `font-size: 16px;` |
| `font-family` | Sets the font type | `font-family: Arial, sans-serif;` |
| `font-weight` | Sets text thickness | `font-weight: bold;` |
| `text-align` | Aligns text inside element | `text-align: center;` |
| `text-decoration` | Adds underline, line-through, etc. | `text-decoration: underline;` |
| `line-height` | Sets spacing between lines | `line-height: 1.5;` |
| `letter-spacing` | Adjusts space between letters | `letter-spacing: 2px;` |
| `text-transform` | Capitalize or uppercase text | `text-transform: uppercase;` |
| `text-shadow` | Adds shadow to text | `text-shadow: 2px 2px #ccc;` |

## 2Ô∏è‚É£ Box Model Properties

| Property | Description | Example |
|----------|------------|---------|
| `width` | Element width | `width: 200px;` |
| `height` | Element height | `height: 150px;` |
| `margin` | Space outside element | `margin: 20px;` |
| `padding` | Space inside element | `padding: 15px;` |
| `border` | Border around element | `border: 2px solid black;` |
| `border-radius` | Rounded corners | `border-radius: 10px;` |
| `box-sizing` | Controls box-sizing method | `box-sizing: border-box;` |
| `box-shadow` | Shadow outside element | `box-shadow: 2px 2px 5px #888;` |

---

## 3Ô∏è‚É£ Background Properties

| Property | Description | Example |
|----------|------------|---------|
| `background-color` | Sets background color | `background-color: #f4f4f4;` |
| `background-image` | Sets background image | `background-image: url('bg.jpg');` |
| `background-size` | Adjusts image size | `background-size: cover;` |
| `background-repeat` | Repeat or no-repeat | `background-repeat: no-repeat;` |
| `background-position` | Position of image | `background-position: center;` |
| `background-clip` | Area background covers | `background-clip: padding-box;` |
| `background-origin` | Where background starts | `background-origin: border-box;` |

## 4Ô∏è‚É£ Display & Layout Properties

| Property | Description | Example |
|----------|------------|---------|
| `display` | Defines element display type | `display: block;` |
| `position` | Element positioning | `position: relative;` |
| `top`, `right`, `bottom`, `left` | Position offsets | `top: 10px;` |
| `float` | Floats element left/right | `float: left;` |
| `clear` | Clears floated elements | `clear: both;` |
| `overflow` | Handles overflow content | `overflow: hidden;` |
| `z-index` | Layer order | `z-index: 10;` |

---

## 5Ô∏è‚É£ Flexbox Properties

| Property | Description | Example |
|----------|------------|---------|
| `display: flex` | Enables flex container | `display: flex;` |
| `flex-direction` | Row or column layout | `flex-direction: column;` |
| `justify-content` | Align items horizontally | `justify-content: center;` |
| `align-items` | Align items vertically | `align-items: flex-start;` |
| `flex-wrap` | Wrap items to next line | `flex-wrap: wrap;` |
| `gap` | Space between flex items | `gap: 10px;` |
